TT WORKSHOP - Static website for GitHub Pages

Cách dùng:
1. Upload toàn bộ thư mục này lên GitHub repository.
2. Vào Settings > Pages > Deploy from branch > chọn main/root.
3. Nếu có video/ảnh thật, thay các khối .media placeholder bằng thẻ <img> hoặc <video>.
4. Link Google Maps hiện đang dùng dạng tìm kiếm theo địa chỉ. Khi có link Google Maps chính xác, thay toàn bộ link maps trong index.html.

File chính:
- index.html
- style.css
- script.js
- assets/ttworkshop.jpg
