# TT WORKSHOP Website

Website tĩnh chạy trên GitHub Pages.

## Cách dùng
1. Upload toàn bộ file lên repository GitHub.
2. Vào Settings > Pages > Deploy from branch.
3. Chọn branch `main` và thư mục `/root`.
4. Mở link GitHub Pages.

## File chính
- `index.html`
- `style.css`
- `script.js`
- `assets/ttworkshop.jpg`
